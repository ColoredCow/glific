    Glific.Jobs.ChatbaseWorker (Glific v1.1.5) View Source

    Process the message table for each organization. Chunk number of messages in groups of 128 and create a Chatbase Worker Job to deliver the message to the chatbase servers

    We centralize both the cron job and the worker job in one module
